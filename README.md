# homebrew-brew

Brew Formulae for my stuff

```sh
brew tap joshrotenberg/brew
```

## Available Formulas

### [jpx](https://github.com/joshrotenberg/jmespath-extensions) - JMESPath CLI with 150+ extended functions

A powerful jq alternative with 189 additional functions for strings, arrays, dates, hashing, encoding, geo, and more.

```sh
brew install jpx
```

### [adrs](https://github.com/joshrotenberg/adrs) - Architectural Decision Record CLI

```sh
brew install adrs
```

### [mdbook-lint](https://github.com/joshrotenberg/mdbook-lint) - Fast markdown linter for mdBook projects

```sh
brew install mdbook-lint
```

### [redisctl](https://github.com/joshrotenberg/redisctl) - Redis CLI tool

```sh
brew install redisctl
```
